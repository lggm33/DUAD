Add Task Endpoint

To add a task, make a POST request with a JSON body containing the following fields:

{
    "title": "string",
    "description": "string",
    "state": "string (one of: 'Por Hacer', 'En Progreso', 'Completada')"
}

Field Requirements:

  • title: Task title (string)
  • description: Task description (string)
  • state: Must be one of:
    • Por Hacer
    • En Progreso
    • Completada

Note: The ID will be automatically assigned by the system.
